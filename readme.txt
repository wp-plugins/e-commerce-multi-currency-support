== Description ==

e-Commerce Multi Currency support is a Add-on Currency Widget that allows your customers to view prices of products and Cart in the Currency they desire, When proceeding to checkout Currency Converter then Changes the Currency Back to the shops base currency, and stores it in Wp-admin as both Base Currency and Currency Viewed by Customer. (requires Wp-e-Commerce 3.8.x)

== Installation ==

1. Upload the folder 'wp-e-commerce-multi-currency-magic' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
